## Recommended Build Order

Van's recommends the order above (HS → Elevators → VS → Rudder), but in practice many builders do VS and Rudder before or alongside the elevators since they are simpler. The HS is always first — it teaches the core skills.

## Cross-cutting Topics

- **Priming:** See [priming.md](../priming.md) — currently using Stewart Systems EkoPoxy (water-based, non-toxic). All non-Alclad parts (HS-609PP, HS-810-1, HS-814-1) **must** be primed before riveting.
- **Rivet gauge:** Always verify rivet length with a rivet gauge before driving. The HS front spar uses AN426AD4-7 (flush) at the center — easy to grab the wrong length.
- **Edge distance:** Minimum edge distance is 2× rivet diameter from hole center to edge. The HS-00005 rib and HS-810/HS-814 angle area is the most common place to violate this. See [horizontal-stabilizer.md](horizontal-stabilizer.md).
- **Fluting ribs:** Flute ribs before match-drilling to skin. If you drill first and then flute, the holes will no longer align. Lesson learned the hard way on the HS.
- **Section 5 references:** The manual constantly refers back to Section 5 for techniques. Key ones:
  - 5.1 — Priming
  - 5.2 — Edge finishing, deburring
  - 5.4 — Riveting
  - 5.5 — Countersinking and dimpling
  - 5.6 — Back riveting
  - 5.7 — Folded trailing edges
  - 5.9 — Rolled leading edges
  - 5.13 — Fluting and straightening ribs

## Tools Needed for Empennage

- Pneumatic or hand squeezer (most rivets)
- C-frame dimpling tool (skins)
- Rivet gun + bucking bar (some locations)
- #30 and #40 drill bits
- Unibit (step drill) — for relief holes and enlarging
- Cleco pliers + clecos (lots of them)
- Hand seamer
- Vixen file + Scotchbrite wheel
- Rivet gauge
- Fluting pliers
- Cardboard template material (for bend angle checks)
